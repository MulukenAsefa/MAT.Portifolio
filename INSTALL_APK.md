# 📱 Install APK on Your Phone - Simple Guide

## 🎯 Your App is Being Built!

The APK file is being created right now. Once complete, follow these steps:

---

## 📍 **Step 1: Find the APK File**

After the build completes, your APK will be at:

```
C:\portifolio\cybersecurity_portfolio\build\app\outputs\flutter-apk\app-release.apk
```

**File name:** `app-release.apk`
**Size:** ~20-30 MB

---

## 📲 **Step 2: Transfer APK to Your Phone**

Choose one method:

### **Method A: USB Cable** (Fastest)
1. Connect phone via USB
2. Open phone in File Explorer
3. Copy `app-release.apk` to phone's `Downloads` folder
4. Done!

### **Method B: Email**
1. Email the APK to yourself
2. Open email on your phone
3. Download the attachment
4. Done!

### **Method C: Cloud Storage**
1. Upload APK to Google Drive/Dropbox
2. Open Drive/Dropbox on phone
3. Download the file
4. Done!

### **Method D: Bluetooth**
1. Right-click APK file
2. Send to → Bluetooth device
3. Select your phone
4. Accept on phone

---

## 🔓 **Step 3: Enable Installation from Unknown Sources**

**First time only:**

1. When you try to install, you'll see:
   ```
   "For your security, your phone is not allowed to install
    unknown apps from this source"
   ```

2. Tap **"Settings"**

3. Enable **"Allow from this source"**

4. Go back and tap **"Install"**

**OR manually enable:**
- Settings → Security → Unknown Sources → Enable
- Or: Settings → Apps → Special Access → Install Unknown Apps → Select your file manager → Allow

---

## ✅ **Step 4: Install the App**

1. **Open** the APK file on your phone
   - Go to Downloads folder
   - Tap `app-release.apk`

2. **Tap "Install"**

3. **Wait** for installation (10-20 seconds)

4. **Tap "Open"** or find the app icon

## 🐛 **Troubleshooting**

### **"App not installed"**
- Make sure you enabled "Unknown Sources"
- Try uninstalling old version first
- Restart phone and try again

### **"Parse error"**
- APK file might be corrupted
- Re-download or re-transfer
- Make sure transfer completed fully

### **"Installation blocked"**
- Go to Settings → Security
- Enable "Unknown Sources" or "Install Unknown Apps"
- Try installation again

### **App crashes on launch**
- Make sure Android version is 5.0 or higher
- Clear app cache: Settings → Apps → Your App → Clear Cache
- Reinstall the app
